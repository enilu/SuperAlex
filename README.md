# 晨光冲锋队 - 儿童晨间流程激励游戏

## 项目概述

"晨光冲锋队"是一款专为儿童设计的H5小游戏，旨在通过游戏化方式引导孩子自主完成早晨流程任务，培养良好的时间管理习惯和责任感，同时减少家长的催促，使晨间时光更加愉快。

## 演示地址
http://superalex.enilu.cn/
## 核心功能

### 1. 任务管理系统
- 支持自定义晨间任务流程（起床、穿衣、刷牙洗脸、吃早餐等）
- 智能匹配当前时间，显示对应任务
- 每个任务配有倒计时和完成状态指示
- 任务完成后自动进入下一任务阶段

### 2. 语音播报系统
- 使用Web Speech API实现中文语音播报
- 根据任务完成情况生成不同的反馈语音
- 支持提前完成、准时完成、超时完成等多种场景的语音反馈

### 3. 成就与激励系统
- 记录每日任务完成情况和闪电完成次数
- 跟踪连续完成任务的天数
- 自动解锁成就勋章，提供正向激励
- 可视化展示成就进度和历史记录

### 4. 音效反馈
- 为各种操作提供音效反馈（按钮点击、任务完成等）
- 支持音效开关控制

### 5. 数据持久化
- 使用localStorage保存游戏数据和成就记录
- 支持重置周数据功能

### 6. 响应式设计
- 适配各种移动设备屏幕尺寸
- 针对儿童使用习惯优化的界面设计
- 支持iOS安全区域适配

## 技术栈

- HTML5 + CSS3 (响应式设计)
- JavaScript (原生JS，模块化结构)
- Web Speech API (语音合成)
- Web Audio API (音效管理)
- localStorage (本地数据存储)

## 安装与部署

1. **克隆项目**

   ```bash
   git clone https://github.com/your-username/morning-champion.git
   cd morning-champion
   ```

2. **部署方式**

   - **静态网站托管**：将项目文件部署到任何支持静态网站的服务器或托管服务（如Nginx、Apache、GitHub Pages、Netlify等）
   - **本地测试**：直接在浏览器中打开`index.html`文件

3. **HTTPS要求**

   > 注意：由于浏览器安全策略，Web Speech API在部分浏览器中需要HTTPS环境才能使用。建议使用HTTPS部署以获得最佳体验。

## 使用方法

### 儿童端使用

1. 访问游戏页面
2. 首次使用会显示引导界面，点击"开始游戏"进入主界面
3. 界面会自动显示当前应该完成的任务
4. 完成任务后，点击"✅ 我完成了！"按钮进入下一任务
5. 所有任务完成后，会显示庆祝界面

### 家长配置指南

目前的简化版本中，任务配置通过`config.js`文件进行管理：

1. 打开`assets/js/config.js`文件
2. 修改`defaultConfig`对象中的任务设置：
   - 设置任务名称、图标、开始时间和截止时间
   - 可调整语音模板内容
   - 可设置每周哪几天启用

### 数据管理

- 长按状态栏可重置本周数据
- 所有数据存储在浏览器的localStorage中，清除浏览器数据会导致游戏数据丢失

## 注意事项

1. **浏览器兼容性**
   - 推荐使用最新版本的Chrome、Firefox、Safari或Edge浏览器
   - iOS设备上，语音播报可能需要用户与页面交互后才能正常工作

2. **后台执行限制**
   - 部分移动设备浏览器在锁屏或后台运行时会限制JavaScript执行
   - 建议在听到提醒后主动打开页面操作

3. **语音权限**
   - 首次使用时，浏览器可能会请求语音合成权限，请允许权限以获得完整体验

## 开发说明

### 自定义配置

您可以通过URL参数覆盖默认配置：

```
https://your-deploy-url.com/?tasks=自定义任务配置
```

### 添加新成就

要添加新的成就类型：

1. 在`achievementSystem.js`文件中的`AchievementType`枚举中添加新类型
2. 在`achievements`数组中定义成就的具体信息和解锁条件
