# 自定义音效说明

## 音效包结构

本游戏支持自定义音效包功能，您可以按照以下结构组织您的音效文件：

- `sounds/default/` - 默认音效包（游戏启动时使用）
- `sounds/video1/` - 自定义音效包1
- `sounds/video2/` - 自定义音效包2

## 支持的音效类型

游戏支持以下音效类型，您需要为每个音效包创建对应的文件：

1. `success.mp3` - 任务完成音效
2. `celebration.mp3` - 庆祝音效
3. `warning.mp3` - 警告音效
4. `tick.mp3` - 倒计时滴答声
5. `complete.mp3` - 确认完成按钮音效

## 添加自定义音效

1. 准备好您想要使用的音效文件（MP3格式最佳）
2. 将音效文件重命名为上述指定的文件名
3. 将文件复制到您想要的音效包目录中（例如 `sounds/video1/`）
4. 在游戏设置中切换到您的自定义音效包

## 注意事项

- 音效文件大小建议不超过1MB，以确保快速加载
- 音效长度建议控制在3-5秒内，避免过长的音效影响游戏体验
- 如果某个音效文件不存在，游戏将使用内置的Web Audio API生成的音效作为后备
