# 默认音效包

本目录包含游戏默认使用的音效文件。由于无法直接生成MP3文件，您需要自行准备以下音效文件：

## 需要的音效文件

1. `success.mp3` - 任务完成音效：简短、积极的提示音
2. `celebration.mp3` - 庆祝音效：欢快、鼓舞人心的音乐片段
3. `warning.mp3` - 警告音效：提醒用户的警示音
4. `tick.mp3` - 倒计时滴答声：清晰、短促的计时音效
5. `complete.mp3` - 确认完成按钮音效：点击确认的反馈音

## 如何获取音效

1. **免费音效库**：
   - [Freesound](https://freesound.org/)
   - [Zapsplat](https://www.zapsplat.com/)
   - [Pixabay](https://pixabay.com/sound-effects/) 最好用

2. **音效生成器**：
   - 使用在线音效生成器创建简单的音效
   - 使用Audacity等免费音频编辑软件

3. **注意事项**：
   - 确保音效长度控制在3-5秒内
   - 文件大小建议不超过1MB
   - 如未提供音效文件，游戏将使用内置的Web Audio API生成音效

请将下载或创建的音效文件重命名为上述指定名称，并放置在此目录中。
